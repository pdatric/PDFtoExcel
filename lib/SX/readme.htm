Feature

The SmartXLS  allows developer to automate any manual spreadsheet process within their organization.

Automate the delivery of Excel reports and give your users the data they need in the format they want.

Web-enable your existing spreadsheet business logic as a scalable server-side calculation engine.

enables you to integrate Excel-compatible spreadsheet functionality in your Windows applications without relying on Excel or writing VBA code.

List of Files

  • Library files
    • SX-4.x.jar -  the SmartXLS library.
  • Documentation, Javadoc
    • readme.html - This document.
    • javadoc - SmartXLS  API help in JavaDoc HTML format.
    • template- template files used in the sample.

Examples Included in the package

The example files are located in the "Examples" directory of the installed files.

  • Formatting - This directory contains an application that demonstrates how to use the RangeStyle object to format a spreadsheet.
  • Formulas - This directory contain sample codes for create formulas.
  • Charts - This directory contain sample codes for create and modify charts.
  • Image - This directory contain sample codes for read/write image to workbook.
